說明 :

執行設定檔 crontrigger.ini
-------------------------------------------------------------------------------
以空格為區分, 每行為單一任務
以 # 字為行開頭則該行不載入 (可寫注解或關閉程序)
參數之間以空格區分, 每隔代表的意義如下
 
 秒  分  時  日  月  週  程序

-------------------------------------------------------------------------------
參數範圍 :
	秒		0~59
	分		0~59
	時		0~23
	日		0~31
	月		0~12
	週		0~6		(0 = 日, 6 = 六)
	程序	程式(.php)檔案名稱
-------------------------------------------------------------------------------
可使用替代字元:

				秒  分  時  日  月  週
 數字			OK	OK	OK	OK	OK	OK
 數字-數字		OK	OK	OK	OK	OK	OK
 *				OK	OK	OK	OK	OK	OK
 *+數字		OK	OK	NT	NT	NT
 */數字		OK	OK	NT	NT	NT

	OK = 可用, NT = 可能(未測試)
-------------------------------------------------------------------------------
特性 :
	啟動時不會執行程序, 程序會在下一個排程時間點才執行

 數值設定為下, 以[秒]作為範例
 *		立即啟動
 12	當時間秒數為12時啟動
 */10	當時間秒數為0,10,20,30,40,50時啟動
 *+3	三秒過後啟動
 6-8	當時間於6,7,8時啟動

 可複合使用, 請在合理範圍內設定
-------------------------------------------------------------------------------
舉例 :
	預定時間		秒  分  時  日  月  週  程序		
	--------------	--	--	--	--	--	--	-----
	每小時			0	0	*	*	*	*	a.php
	3~9點每分鐘		0	*	3-9	*	*	*	b.php
	每五分鐘		*	*/5	*	*	*	*	c.php
	兩小時後		*	*	*+2	*	*	*	d.php
	每天12點15分	0	15	12	*	*	*	e.php
-------------------------------------------------------------------------------
附註 :
	* 本程序條件過於複雜可能會出錯
	* 如改變設定值必須重新啟動
	* 如需進位, 例如跨天, 請設定兩組
	* 無判斷閏年
-------------------------------------------------------------------------------
可能發生問題 :
	* 行為模式跨天
